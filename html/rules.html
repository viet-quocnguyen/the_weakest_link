Game Rules

Game is 3 rounds

Round 1:

  1. Round is 2 min long
  2. Correct answers move player up money tree
  3. Player has option to bank money won
  4. If player answers incorrectly they go back to banked amount
  5. Otherwise it goes back to zero
  6. Max that can be banked is $500,000

Round 2:

  1. Money brought over from Round 1
  2. Similiar rules as Round 1
  3. Round is 90 seconds long
  4. Max that can be banked is $1,000,000
  5. Can only move onto Round 3 if money was banked otherwise you lose!

Round 3:

  1. Round is 1 min long
  2. Player must answer all questions within time limit
  3. Incorrect answer results in a loss of everything!
